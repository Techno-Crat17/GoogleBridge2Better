# Bridge2Better AI

Bridge2Better AI is a production-grade AI-powered reverse mentorship platform.
It connects underserved youth with retired professionals for bidirectional learning.

## Tech Stack
- Frontend: Next.js 15, TailwindCSS, shadcn/ui
- Backend: FastAPI, PostgreSQL, Redis
- AI Services: HuggingFace, LangChain, Qdrant
- Infrastructure: Docker, Docker Compose

## Quick Start
1. Copy `.env.example` to `.env`
2. Run `docker-compose up --build`
3. Access Frontend at `http://localhost:3000`
4. Access Backend API docs at `http://localhost:8000/docs`
5. Access AI Services API docs at `http://localhost:8001/docs`
